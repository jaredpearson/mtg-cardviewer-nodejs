
# Starting on local machine

```
foreman start
```

# Starting in Vagrant VM

```
vagrant up
vagrant ssh
cd app
npm start
```

# Getting the data
To update the card data, use the `sync` script as shown below. Make sure to check that the set symbol images for any added sets are correct.

```
npm run sync
```
